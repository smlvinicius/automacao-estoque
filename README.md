# 📊 Automação de Relatório de Itens Sem Venda

## 🚀 Sobre o projeto

Este projeto nasceu de uma necessidade real do meu dia a dia profissional.

Atualmente atuo com análise de estoque e identifiquei uma oportunidade de melhoria em um processo manual que consumia entre **30 a 40 minutos diariamente** para gerar relatórios de itens sem venda por setor (corredor).

Buscando aplicar na prática os conhecimentos que venho desenvolvendo na área de dados, desenvolvi uma automação utilizando Python para otimizar esse processo.

---

## ⚡ Problema

O processo era totalmente manual e envolvia:

* Aplicar filtros no Excel
* Identificar itens sem venda
* Separar informações por setor
* Analisar corredor por corredor

Além de consumir tempo, o processo estava sujeito a erros e falta de padronização.

---

## 💡 Solução

Desenvolvi um script em Python utilizando Pandas que:

* Lê automaticamente o arquivo Excel
* Aplica regras de negócio definidas
* Filtra itens sem venda
* Organiza os dados por setor
* Gera um relatório final estruturado com abas separadas

---

## 📉 Regras de negócio aplicadas

* Dias sem venda ≥ 7
* Idade ≥ 7

---

## ⏱️ Resultado

* Tempo reduzido de **30–40 minutos para menos de 1 minuto**
* Processo padronizado
* Redução de erros manuais
* Maior agilidade na análise dos dados

---

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Excel
* Google Colab
