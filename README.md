# 📊 Automação de Relatório de Itens Sem Venda

## 🚀 Contexto
No meu dia a dia como analista de estoque, eu precisava gerar manualmente um relatório de itens sem venda por setor.

Esse processo levava entre **30 a 40 minutos diariamente**.

---

## ⚡ Solução
Desenvolvi uma automação utilizando Python e Pandas que:

- Aplica regras de negócio automaticamente  
- Filtra itens sem venda  
- Organiza os dados por setor  
- Gera um relatório em Excel pronto para uso  

---

## 📉 Regras aplicadas
- Dias sem venda ≥ 7  
- Idade ≥ 7  

---

## ⏱️ Resultado
O tempo de execução foi reduzido de **40 minutos para menos de 1 minuto**.

---

## 🛠️ Tecnologias
- Python  
- Pandas  
- Excel
